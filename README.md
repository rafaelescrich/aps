## INSTALL

### REQUIREMENTS

- Maven
- Java 1.8 (not tested in older versions yet)

### DOWNLOAD, COMPILE AND RUN (using Maven)

```
git clone [this repo URL]
cd [cloned-dir]
mvn compile
mvn exec:java
```
